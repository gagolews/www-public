genieclust 1.0.1 on PyPI and CRAN

A new release of genieclust has been published on PyPI and CRAN.

Changes since v1.0.0:

  • [GENERAL] A paper on the genieclust package has appeared in SoftwareX.

  • [Python] plot_scatter now uses a more accessible default palette (from R 4.0.0).

  • [Python] New function: inequity.devergottini_index.

  • [R] New function: devergottini_index.
